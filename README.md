# Application-d-Analyse-de-Sentiment
Brief N°6 : Une API FastAPI sécurisée (JWT) qui envoie des textes à l’API Hugging Face pour analyser leur sentiment avec le modèle nlptown/bert-base-multilingual-uncased-sentiment. L’API doit retourner une note de 1 a 5. Le service doit etre propre, fonctionnel, conteneurisé avec Docker ,accompagné d’un frontend en Next.js et testé via Postman
